Приложение на tkinter

Используем модель проектировани MVC
Способ хранения данных OOP

Используемые виджеты tkinter:

- Frame
- Label
- Button
- Entry

Используется метод bind(), для связи:
widget(виджет)-Event(событие)-Funkction(функция)

widget.bind(event, fucxtion)

Приложение по умолчанию конвертирует:
километры в милли (1 km = 1.6 mi )
